Frameworks

Herramientas de apoyo para desarrollo web.

Django

Es un framework web de Python de alto nivel que fomenta un desarrollo rápido y un diseño limpio y pragmático. Creado por desarrolladores experimentados, se encarga de gran parte de las molestias del desarrollo web, por lo que puede concentrarse en escribir su aplicación sin necesidad de reinventar la rueda. Es gratis y de código abierto.

Flask

Es un framework minimalista escrito en Python que permite crear aplicaciones web rápidamente y con un mínimo número de líneas de código. Está basado en la especificación WSGI de Werkzeug y el motor de templates Jinja2 y tiene una licencia BSD.

Laravel

Es un framework de aplicación web con una sintaxis expresiva y elegante. Ya sentamos las bases, liberándote para crear sin preocuparte por las cosas pequeñas.
